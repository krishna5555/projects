**Expense Tracker**

**Functionality:**
Expense tracker app can be used to:
* Manage expenses and income
* Manage bill reports
* View Expense reports
* Delete expense reports

**Tech Stack used:**
Spring-Boot, Spring Data-JPA, React JS, Docker, Kubernetes
